# ⚔️ Ranked Match Calculator

DIO Logic Fundamentals Challenge — Calculates a player's rank based on wins and losses.

## Ranks

| Wins | Rank |
|------|------|
| < 10 | Ferro |
| 10–19 | Bronze |
| 20–49 | Prata |
| 50–79 | Ouro |
| 80–89 | Diamante |
| 90–99 | Lendário |
| 100+ | Imortal |

## Usage
```bash
node index.js
```

Or open `index.html` in a browser for the interactive UI.

## Stack

- Vanilla JavaScript (Node.js + Browser)
- No dependencies
